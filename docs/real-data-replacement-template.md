# 真实数据替换结构模板

## 目标

将当前演示结构中的策略与基准对比字段替换为真实、可追溯、可说明的数据。

## 建议真实数据字段

每个产品/策略至少准备：
- 产品名称
- 产品代码
- 统计区间
- 策略收益率
- 基准名称
- 基准收益率
- 最大回撤
- 基准最大回撤
- 波动水平或年化波动率
- 更新时间
- 数据来源
- 备注说明

## 当前页面建议显示字段

- 策略收益率
- 基准收益率
- 最大回撤
- 基准回撤
- 波动水平
- 统计区间
- 更新时间
- 数据来源

## 替换要求

- 如果尚未接入真实数据，必须明确标注“演示结构，待替换为真实统计结果”
- 不得让示意数据在领导展示场景下被误认为真实正式业绩披露
- 后续建议由统一数据源生成，不手工散落填写
