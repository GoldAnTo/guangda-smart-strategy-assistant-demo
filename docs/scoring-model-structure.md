# 推荐评分模型结构稿

## 目标

将当前的总匹配分升级为“可拆解、可解释、可展示”的评分模型。

## 建议总分结构

```text
总分 = 风险匹配分 + 期限匹配分 + 流动性匹配分 + 收益目标匹配分 + 资产平衡分
```

## 分项权重建议

- 风险匹配分：30 分
- 期限匹配分：20 分
- 流动性匹配分：20 分
- 收益目标匹配分：20 分
- 资产平衡分：10 分

## 分项说明

### 1. 风险匹配分（30）
- 客户风险偏好与产品风险等级的适配程度
- 这是最高权重，优先保障边界安全

### 2. 期限匹配分（20）
- 客户投资期限与产品期限标签匹配程度

### 3. 流动性匹配分（20）
- 客户流动性要求与产品流动性特征匹配程度

### 4. 收益目标匹配分（20）
- 客户目标为稳健、增厚、平衡还是定制
- 产品定位是否对应客户目标

### 5. 资产平衡分（10）
- 若客户已有较多权益资产，新增稳健资产可加分
- 若客户当前配置已偏保守，继续配置相似方向则不额外加分

## 后端建议输出字段

每个推荐候选可扩展为：

```json
{
  "productId": "fsplus-001",
  "productName": "稳盈固收增强1号",
  "matchScore": 92,
  "scoreBreakdown": {
    "riskScore": 30,
    "durationScore": 18,
    "liquidityScore": 18,
    "goalScore": 18,
    "balanceScore": 8
  }
}
```

- 总匹配分：92
- 风险匹配：30/30
- 期限匹配：18/20
- 流动性匹配：18/20
- 收益目标匹配：18/20
- 资产平衡：8/10
